More About me

Code has always been present

I have been playing with HTML and CSS since I got the internet in the 90s, from my first web page provided by my internet provider to the customisable pages on myspace. I later started tinkering with WordPress themes and have been using websites associated with projects or jobs I had in the video Industry. Parallel to all of this, I was always interested in the arts, more as a hobby, and started to explore generative art.

Art taught me how to program

I started with an interest in art, but my interest grew in creating systems. I learned concepts like OOP or Recursion without knowing I was learning them.

Programing becomes the main goal

It would be a short time until I surrendered to programming. And the joy was more in solving problems than the results in themselves.

Becoming industry ready

In 2022 I decided to take this further and join Code Institute. I wanted to work in the industry, so I wanted to make sure that not only was I programming correctly, but I knew and understood the industry ways.

Madyard

Description

Madyard is a plant business I helped to set up that sells plants from Madeira (my home town) and Canaries in the UK. My partner is very computer challenged, so the website had to be easy to update the never-ending ins and out of plant stock. I used Isotope JS to create a dynamic plant catalogue that you can sort and filter according to several categories.
